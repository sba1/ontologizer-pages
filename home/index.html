Ontologizer

Ontologizer is a tool for the statistical analysis and visualization of high-throughput biological data using Gene Ontology. Most conveniently, it can be started via the Java Webstart facility:

Note however that the Webstart facility will no longer work by default with recent versions of the Java runtime due to increased security settings. See the Webstart section for details.
